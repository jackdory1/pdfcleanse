QUICK START
-----------
1. Open pdf-cleanse.html in Chrome, Firefox, Edge, or Safari.
2. Drag and drop one or more PDF files onto the upload area.
3. Click "Strip metadata & download."
4. Download the audit certificate for your compliance records.

No installation. No internet required after first load.
No files are uploaded anywhere. Ever.

DEPLOYMENT OPTIONS
------------------
  Browser (local):   Double-click pdf-cleanse.html
  Intranet:          Copy to any web server directory — no server config needed
  Air-gapped:        Self-host pdf-lib CDN script (see whitepaper Section 5)
  USB distribution:  Copy this entire folder to a USB drive

VERIFICATION
------------
After cleaning, always verify the output in:
  Adobe Acrobat → File → Properties → Description (and Custom)
This confirms all visible metadata fields have been cleared.
